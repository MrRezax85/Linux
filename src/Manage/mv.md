# دستور mv

جابجایی (انتقال) فایل یا دایرکتوری.

**استفاده:**

</div>

> mv [منبع] [مقصد]

<div dir="rtl">

**مثال‌ها:**

جابجایی یک فایل به مسیر دیگر:

</div>

> mv file.txt /path/to/destination

<div dir="rtl">

جابجایی یک دایرکتوری به مسیر دیگر:

</div>

> mv directory /path/to/destination
